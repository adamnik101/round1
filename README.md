# Round 1 Application

Solution done with ASP.NET Core Web API and Angular.



## Installation & configuration

Run if needed

```bash
  cd RareCrewAngularAPP
  npm install
```
Start Angular application
```bash
  ng serve
```
Start ASP.NET application
```bash
  cd RareCrew.API
  dotnet run
```
## Configure ports

Angular
```bash
  constants/server.ts
```
ASP.NET Core Web API
```bash
  appsettings.json
```
